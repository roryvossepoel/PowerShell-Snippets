# PowerShell Snippets

A collection of PowerShell scripts focused on:

- Microsoft Intune
- Windows Autopilot
- Microsoft Entra ID
- Windows Management
- Automation

## Available Scripts

### CompanyPortalLauncher

Launches Microsoft Company Portal automatically after the first user sign-in and removes itself afterwards.

| Script | Description |
|----------|----------|
| Launch-CompanyPortalAfterFirstLogin.ps1 | Creates a one-time startup shortcut that launches Company Portal after the user's first desktop session. |
